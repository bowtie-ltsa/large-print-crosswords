Welcome to Large Print Crosswords

Large-Print-Crosswords (LPC) fetches crossword puzzles from the interwebs and formats them for printing in large fonts while still fitting them onto a single printed page.

  • Produces HTML so you can just use your browser to do the printing.
  • Can fetch several puzzles with one click, so you can print several pages in one go.
  • Currently limited to just one puzzle: The Universal Crossword.
  • Currently making assumptions about page size.

Instructions:

  1. Select a date for the puzzle you want to print.
  2. Click the "Fetch Puzzles" button.
  3. Wait for the puzzles to be fetched and formatted.
  4. Print the puzzles using your browser's print functionality.
